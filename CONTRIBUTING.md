# Contributing to Driver Drowsiness AI

First off, thank you for considering contributing to this project! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

### 1. Reporting Bugs
If you find a bug, please create a new issue in the "Issues" tab. Be sure to include:
* A clear description of the bug.
* Steps to reproduce it.
* Your operating system and Python version.

### 2. Suggesting Enhancements
Have an idea to make the AI better? Open an issue and tag it as an "enhancement."

### 3. Pull Requests (PRs)
We welcome Pull Requests! Here is the workflow:
1. **Fork** the repository.
2. Create a new branch: `git checkout -b feature/AmazingFeature`.
3. Make your changes.
4. **Format your code:** We use `black` for code styling. Please run `black .` before committing.
5. Commit your changes: `git commit -m 'Add some AmazingFeature'`.
6. Push to the branch: `git push origin feature/AmazingFeature`.
7. Open a **Pull Request**.
